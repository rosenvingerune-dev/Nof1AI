## üîç Troubleshooting

### Problem: "Insufficient balance"

**Symptom:**
```
‚ùå No testnet funds found!
   Balance: $0.00 USDC
```

**Solution:**
1. Check that you sent Discord command:
   ```
   !faucet YOUR_ADDRESS
   ```
2. Wait 30-60 seconds
3. Run balance check again
4. If still $0:
   - Try Discord command again
   - Ask in #testnet-support channel

### Problem: "Invalid private key"

**Symptom:**
```
‚ùå Failed to initialize API: Invalid private key format
```

**Solution:**
1. Check that private key starts with `0x`
2. Verify that there are 66 characters total (0x + 64 hex chars)
3. No spaces or line breaks
4. Generate new wallet if corrupt:
   ```bash
   python scripts/setup_hyperliquid_testnet.py
   ```

### Problem: "Connection failed"

**Symptom:**
```
‚ùå Failed to fetch balance: Connection error
```

**Solution:**
1. Check internet connection
2. Verify that `HYPERLIQUID_NETWORK=testnet` in .env
3. Check testnet status:
   - https://status.hyperliquid.xyz
   - Hyperliquid Discord #testnet-status
4. Try again in 5 minutes

### Problem: "Testnet is down"

**Symptom:**
```
‚ö†Ô∏è  Hyperliquid testnet might be down
```

**Solution:**
1. Go to Discord #testnet-status
2. Check if others have same problem
3. Wait for testnet to come back up
4. Testnet is not 100% uptime guaranteed
5. Can continue with other tests (Gemini API etc.)
